# 🚀 Rescue Nexus 🌍  
**A Crowdsourced Disaster Management Platform**  

## 📌 About the Project  
**Rescue Nexus** is a web-based platform designed to enhance disaster management efforts by connecting affected individuals with emergency response teams, volunteers, and NGOs. This platform leverages crowdsourcing, real-time mapping, and communication tools to provide faster relief in crisis situations.  

## 🔥 Key Features  
✅ **Real-time Disaster Mapping** – Interactive maps showing disaster locations and affected areas.  
✅ **Emergency Response Requests** – Users can request assistance and track response teams.  
✅ **Volunteer & NGO Collaboration** – Connects volunteers and NGOs with disaster-affected regions.  
✅ **Resource Distribution Management** – Helps manage food, medical supplies, and shelter allocation.  
✅ **Automated Alerts & Notifications** – Sends real-time updates on disaster relief efforts.  

## ⚙️ Tech Stack  
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** Django, Python  
- **Database:** PostgreSQL / MySQL  
- **APIs:** NASA Weather Data, OpenStreetMap
